# gRPC Inventory Frontend

The frontend client for the gRPC Inventory System, built with **Node.js** and **HTML**.

## Related Project
⚙️ **Backend:** https://github.com/Hanna-adel/grpc-inventory-backend-.git

## Project Structure
- **index.html** - Main UI page
- **index.js** - Client-side logic connecting to the gRPC backend
- **dist/main.js** - Bundled output

## Technologies
- Node.js
- HTML / JavaScript
- gRPC Web
